# Front End Foundations 1

**JavaScript - Scope**

_Global Scope_

* A variable that is declared outside a function definition is a global variable, and its value is accessible and modifiable throughout your program.

_Function Scope_

* Each function creates a new scope. Variables defined inside a function are not accessible from outside the function.

_Block Scope_

* Variables introduced with a block are scoped to the containing function or script, and the effects of setting them persist beyond the block itself.
